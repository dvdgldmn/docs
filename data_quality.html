1 Обязательные поля при регистрации животного

Дата рождения

КРИТИЧНО • 24-36% пусто

Почему обязательно: Возраст — базовый клинический параметр. Без него невозможно:

Возрастные напоминания — автоматические SMS о вакцинации щенков (8, 12, 16 нед.), ежегодной ревакцинации, чекапе для пожилых (7+ лет)
Дозировка препаратов — возраст влияет на расчёт дозы, особенно для щенков и пожилых
Скрининг по возрасту — кардио-чекап для пожилых, ортопедический скрининг для молодых крупных пород
Аналитика — возрастная кривая заболеваний, прогнозы, планирование загрузки клиники
Что откроется: Автоматические напоминания о вакцинации → +15-20% повторных визитов. Возрастной скрининг → раннее выявление болезней → лояльность клиентов.

Порода (конкретная)

ВАЖНО • ~65% «Mixed Breed»

Почему обязательно: Порода определяет предрасположенность к болезням, ожидаемый вес, продолжительность жизни.

Породный чекап — шпицу нужно проверять трахею и сердце, пуделю — глаза и уши, хаски — параанальные железы
Нормы веса — невозможно определить ожирение без нормы для породы
Маркетинг — целевые предложения: «Вашему корги пора на кардиочекап»
Статистика — какие породы болеют чаще, какие приносят больше выручки
Сейчас: 65% собак записаны как «Medium Mixed Breed» — это не всегда реальные метисы. Часто просто не вносят породу. Это убивает всю породную аналитику.

Пол

ВАЖНО • 32% «Unknown»

Почему обязательно: Пол влияет на болезни (пиометра — только самки, простатит — только самцы), дозировку, поведение.

Клинически — 100% информации для диагностики репродуктивных проблем
Предложение стерилизации — нельзя предложить стерилизацию, если не знаешь пол

Статус стерилизации

КРИТИЧНО • 6% собак, 13% кошек

Почему обязательно: Стерилизация — один из ключевых медицинских и коммерческих факторов.

Конверсия в услугу — ~3,200 собак и ~6,700 кошек без записи. Если хотя бы 10% согласятся — это ~1,000 операций
Риск ожирения — стерилизованные набирают вес в 2-3x чаще. Без данных невозможно мониторить
Репродуктивные болезни — пиометра, крипторхизм, опухоли — риск только у нестерилизованных
Повторные визиты — стерилизация → осмотр через 10 дней → снятие швов → контроль веса
Что откроется: Автоматическое предложение стерилизации нестерилизованным → при 10% конверсии это ~80,000-100,000 выручки + последующие визиты.

Вес (в профиле)

ВАЖНО • ~50% пусто

Почему обязательно: Вес — базовый параметр для дозировки, диагностики ожирения, мониторинга здоровья.

Дозировка — точность препаратов, наркоза, инфузий
Ожирение — невозможно выявить без исходного веса и нормы для породы

2 Обязательные поля при каждом визите

Жалоба / причина обращения

КРИТИЧНО • 0% заполнено

Это самое важное незаполняемое поле. Без жалобы невозможно понять, зачем клиент пришёл.

Причины обращений — сейчас мы видим только процедуры (рентген, УЗИ), но не знаем ПОЧЕМУ. «Рентген» — это перелом? Кашель? Проглотил игрушку?
Эпидемиологический мониторинг — всплеск жалоб на рвоту/диарею = возможная вспышка. Сейчас это невидимо
Маркетинг — «Ваша порода часто приходит с проблемами кожи — запишитесь на дерматологический чекап»
Качество лечения — если клиент пришёл с жалобой X и через месяц снова с X — лечение не помогло
Формат: Выпадающий список из 15-20 стандартных жалоб + поле «другое». Заполнение — 5 секунд. Варианты: рвота, диарея, не ест, хромает, чешется, кашляет, выделения из глаз/ушей, травма, плановый осмотр, вакцинация, стерилизация, консультация, другое.

Вес при визите

КРИТИЧНО • ~50% пусто

Почему при каждом визите: Однократный вес бесполезен. Ценность — в динамике.

Раннее выявление — потеря 10% веса за месяц = тревожный сигнал (опухоль, диабет, почки). Набор 15% = ожирение
Автоалерт — система может автоматически флажить: «Этот пёс набрал 12% за 3 месяца — рекомендуем диетическую консультацию»
Контроль лечения — если назначена диета, вес при следующем визите покажет результат
Предиктивная модель — с данными по весу можно строить модели прогнозирования ожирения и метаболических болезней
Как внедрить: Весы на входе в кабинет. Взвешивание — часть протокола приёма. Запись автоматическая (цифровые весы → система).

Температура

ВАЖНО • записывается частично

Почему: Температура — первый индикатор воспаления, инфекции, теплового удара. Норма собаки 37.5-39.0°C, кошки 38.0-39.5°C.

Сортировка — температура выше нормы = приоритетный приём
Аналитика — средняя температура при разных диагнозах, выявление паттернов

Диагноз (стандартизированный)

КРИТИЧНО • все «Preliminary», нет стандарта

Почему: Диагнозы записываются свободным текстом, все помечены «Preliminary». Один и тот же диагноз может быть записан 5-ю способами.

Стандартизация — справочник из 100-200 стандартных диагнозов с кодами. Это позволит автоматически группировать, сравнивать, строить статистику
Статус — Предварительный → Подтверждённый → Снят. Сейчас нет разницы между подозрением и фактом
Хроника — история болезней животного как единый таймлайн, а не разрозненные записи
Отчётность — какие болезни лечим чаще всего, какие рецидивируют, какие породы в зоне риска

BCS — балл кондиции тела (1-9)

НЕ ЗАПИСЫВАЕТСЯ • 0%

Почему: BCS — это золотой стандарт оценки ожирения в ветеринарии. Вес сам по себе ничего не значит без контекста породы и телосложения.

Объективная оценка — BCS 1-3 = истощение, 4-5 = норма, 6-7 = избыток, 8-9 = ожирение
Мониторинг — BCS 7 при визите → назначение диеты → BCS 5 через 3 месяца = успех
Предиктивная модель — BCS + порода + возраст + стерилизация = точный прогноз риска ожирения
Допуслуга — «Диетическая консультация» для BCS ≥7 — новая линия дохода
Как внедрить: Пластиковая карта со шкалой BCS на стене каждого кабинета. Врач оценивает визуально + пальпация. Заполнение — 3 секунды.

3 Желательные поля (следующий уровень)

Тип питания

ЖЕЛАТЕЛЬНО

Выпадающий список: сухой корм / влажный корм / смешанный / натуральный / ветеринарная диета / неизвестно.

Зачем: Связь диеты с ожирением, аллергиями, ЖКТ-проблемами. Возможность рекомендовать корма (партнёрство с брендами = доход).

Активность / прогулки (собаки)

ЖЕЛАТЕЛЬНО

Варианты: нет прогулок / 1 раз в день / 2+ раз / активный спорт.

Зачем: Активность — ключевой фактор ожирения и ортопедических проблем. Корреляция с кардио-здоровьем.

Вакцинационный статус

ЖЕЛАТЕЛЬНО

Актуальная / просроченная / не вакцинирован / неизвестно.

Зачем: Автонапоминания о ревакцинации = стабильный поток повторных визитов. По данным клиники, вакцинация уже генерирует 40,000+ выручки.

Источник клиента

ЖЕЛАТЕЛЬНО

Откуда узнал: рекомендация / Instagram / Google / проходил мимо / повторный визит.

Зачем: Понимание каналов привлечения → оптимизация маркетингового бюджета. Сейчас это чёрный ящик.

4 Что откроется при заполнении: дорожная карта

Фаза 1: Базовые поля (1-2 недели)

Внедрить обязательные поля: дата рождения, пол, порода, стерилизация, вес при визите, жалоба.

  • Автоматические SMS-напоминания о вакцинации
  • Предложение стерилизации нестерилизованным
  • Дашборд «причины обращений» в реальном времени
  • Потенциал: +80,000-100,000/год (стерилизация + повторные визиты)

Фаза 2: Мониторинг (1-2 месяца)

Вес при каждом визите + BCS + стандартизация диагнозов.

  • Автоалерты: «Собака набрала 12% веса — рекомендуем диету»
  • Породные чекап-карты: автоматические рекомендации при записи
  • Статистика: топ болезней, рецидивы, эффективность лечения
  • Потенциал: +15-20% среднего чека (дополнительные диагностические услуги)

Фаза 3: Продвинутая аналитика (3-6 месяцев)

Достаточно данных для предиктивных моделей и сегментации.

  • Предиктивная модель ожирения: «Этот шпиц с BCS 6 и набором веса — в зоне риска»
  • Сегментация клиентов: VIP / лояльные / в зоне оттока / потерянные
  • Персонализированные SMS: «Вашему [имя] 7 лет — пора на кардиочекап для [породы]»
  • Потенциал: +10-15% retention → значительный рост LTV

Фаза 4: Экосистема (6-12 месяцев)

Полная аналитическая платформа.

  • Автоматический breed health report для каждого клиента при первом визите
  • Подписка на «план здоровья» для породы — ежеквартальные чекапы
  • Партнёрство с кормовыми брендами на основе данных по диетам
  • Бенчмаркинг: сравнение клиники с отраслевыми стандартами
  • Потенциал: новая бизнес-модель — подписка вместо разовых визитов
