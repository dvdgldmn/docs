🐕 Топ-10 пород собак

Шпиц

328 собак

Проблемы

  • Х-алопеция (породная)
  • Кардиомиопатия
  • Зубной камень / налёт
  • Блошиный дерматит
  • Коллапс/стеноз трахеи
  • Луксация пателлы
  • Ожирение

Сервисы для продажи

  • базовый ЭХО-КГ / ЭКГ ежегодно
  • базовый Чистка зубов / санация
  • допродажа Дерматологический осмотр
  • допродажа Рентген трахеи (если кашель)
  • пакет «Здоровый Шпиц» — сердце+зубы+кожа
  • товар Антипаразитарные средства
  • товар Диетический корм (ожирение)

Апсейл

  • Кардиочекап: ЭХО + ЭКГ + приём = 80-120
  • Стоматологический пакет: чистка + рентген + санация = 60-100
  • Груминг + дерматолог = 40-60

💡 Рекомендации по увеличению среднего чека

1. Породные пакеты — основной драйвер

Создать 5-7 породных пакетов осмотра

«Здоровый Шпиц» (сердце+зубы+кожа), «Здоровый Пудель» (уши+глаза), «Здоровый Британец» (моча+вакцинация), «Брахицефал-чекап» (Пекинес/Мопс — глаза+дыхание+сердце), «Скоттиш-чекап» (суставы+кожа).

+30-50% к чеку при каждом профилактическом визите. При 200 визитах/мес и 20% конверсии в пакет — +2,000-3,000/мес.

2. Допродажа по диагнозу

Автоматические рекомендации в карточке пациента

Если диагноз «дерматит» → предложить аллергопробу + гипоаллергенный корм. Если «зубной камень» → предложить ультразвуковую чистку. Если «отит» → предложить цитологию + ушные капли. Врач видит подсказку на экране.

+15-25% к среднему чеку. Не требует дополнительного маркетинга — работает на существующих визитах.

3. Товары при каждом визите

Антипаразитарные средства — главный товарный драйвер

Блошиный дерматит — в топ-5 проблем у 8 из 10 пород собак и 8 из 10 пород кошек. Каждый визит — повод предложить антипаразитарную обработку (капли, таблетки). Маржа на препаратах высокая.

При 1,000 визитов/мес и 30% конверсии × 10 средний чек на препарат = +3,000/мес.

Диетические корма по показаниям

Пищевая аллергия/непереносимость — в топ проблем у Йорков, Шиншилл, Скоттишей. Партнёрство с Royal Canin / Hills → рекомендация + продажа в клинике.

Маржа 30-40% на кормах + привязка клиента (приходит за кормом = повод для осмотра).

4. Стерилизация как точка входа

Стерилизация → цепочка визитов

Стерилизация/кастрация — не разовая услуга: предоперационные анализы (ОАК + биохимия) → операция → контроль через 3 дня → снятие швов через 10 → контроль веса через 3 мес (риск ожирения).

1 стерилизация = 4-5 визитов × средний чек. Общий доход с цепочки: 120-200 на животное.

5. Возрастные чекапы

Автоматическое предложение по возрасту

7+ лет: кардиочекап + биохимия + УЗИ брюшной. 5+ лет для крупных пород. SMS за 2 недели до дня рождения: «Вашему [имя] исполняется [N] — пора на возрастной чекап для [породы]».

Возрастной чекап = 80-150. При 500 собак 7+ лет и 15% конверсии = 6,000-11,000/год.

6. «Первый визит» — максимальный пакет

Пакет «Новый питомец»

Первый визит — момент максимальной готовности владельца платить. Пакет: полный осмотр + вакцинация + дегельминтизация + чип + рекомендации по породе.

Вместо чека 25-30 (просто осмотр) → 80-120 (пакет). При 80 новых клиентов/мес — +4,000-7,000/мес.
